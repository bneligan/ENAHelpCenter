###Throttle agent You can limit the amount of bandwidth an agent uses during upload and download.

To limit an agent's upload speed:

  1. Go to Actions for that agent on the Agents page.
  2. Click Throttle.
  3. Enter a number in the Maximum upload speed field and select the unit of measure you would like to use from the drop down.
  4. Click Apply.

To limit an agent's download speed:

  1. Go to Actions for that agent on the Agents page.
  2. Click Throttle.
  3. Enter a number in the Maximum download speed field and select the unit of measure you would like to use from the drop down.
  4. Click Apply.
